# SDFproject_GST
A simple program with three user interfaces 
1.Shopkeeper
2.Cashier
3.Loader 

Where we provide the functions ;-
       1. View All Items
       2. Search Item (by Name or Code)
       3. Add New Item
       4. Delete Item
       5. Update Quantity
       6. Generate Bill
       7. Show Low Stock Items
       8. Exit
We are intially starting with 10 stationary items the inventory can be changed any time 

Future plan includes a monthly gst report
